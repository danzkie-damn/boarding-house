# 🏠 Boarding House Manager

A complete boarding house management system built with React.

## Features
- Dashboard with paid/unpaid tenants
- Tenant management with profiles and history
- Monthly billing with auto-generate
- KWH meter reader with auto carry-over
- Invoice generator with SMS copy
- Finance tracking (monthly + yearly)
- Payment history per tenant
- Data backup and restore
- Dark/light mode
- Former tenant records

## Built with
- React + Vite
- GitHub Pages
